# User Guide: Enable Notifications

## 📱 Why This Prompt Appears

The app needs notification permission to send you:
- 💰 Payment reminders
- ⚠️ Service alerts
- 📢 Important account updates

**The prompt will not disappear until you enable notifications.**

---

## ✅ How to Enable (Easy Way)

### Step 1: Click "Enable Notifications Now"
The app will ask Android for permission.

### Step 2: Allow Permission
When Android shows the permission dialog:
- ✅ Click **"Allow"** or **"OK"**
- ❌ Don't click "Block" or "Deny"

### Step 3: Done!
The prompt will disappear and you'll receive notifications.

---

## 🔧 If You Accidentally Clicked "Block"

Don't worry! You can enable it manually:

### Method 1: From App Settings

1. **Long press** the app icon
2. Tap **"App info"** or **ⓘ**
3. Tap **"Permissions"** or **"Notifications"**
4. Enable **"Notifications"**
5. Go back to the app
6. Prompt should disappear!

### Method 2: From Phone Settings

1. Open **Settings** on your phone
2. Tap **"Apps"** or **"Applications"**
3. Find **"ISP Client Portal"**
4. Tap **"Permissions"** or **"Notifications"**
5. Enable **"Notifications"**
6. Return to the app

### Method 3: Reinstall App (Fresh Start)

1. **Uninstall** the app
2. **Install** again
3. **Login**
4. When prompt appears, click **"Allow"**
5. Done!

---

## 📖 Detailed Instructions

### For Samsung Devices:

```
Settings → Apps → ISP Client Portal → Notifications → Allow notifications
```

### For Xiaomi/MIUI:

```
Settings → Apps → Manage apps → ISP Client Portal → Notifications → Allow
```

### For Oppo/ColorOS:

```
Settings → App Management → App list → ISP Client Portal → Notifications → Allow
```

### For Vivo/FuntouchOS:

```
Settings → Apps & notifications → App info → ISP Client Portal → Notifications → Allow
```

### For Stock Android:

```
Settings → Apps → See all apps → ISP Client Portal → Notifications → Allow
```

## ❓ Common Questions

### Q: Why can't I close this prompt?
**A:** Notifications are required for the app to work properly. You must enable them to continue.

### Q: What if I don't want notifications?
**A:** Unfortunately, notifications are required for important account updates. You can disable them later in phone settings if needed.

### Q: I enabled it but the prompt is still there?
**A:** 
1. Close the app completely
2. Open it again
3. The prompt should check and disappear

### Q: The notification toggle is grayed out?
**A:** You're using an old version of the app. Please:
1. Uninstall the app
2. Download and install the latest version
3. Try again

## 🎯 What Happens After Enabling

Once enabled, you'll receive:

### Payment Reminders:
- 3 days before due date
- On due date
- 1 day after due date (if overdue)

### Service Alerts:
- Maintenance notifications
- Service outages
- Network updates

### Account Updates:
- Payment confirmations
- Account status changes
- Important announcements

---

## 🔕 How to Disable Later (If Needed)

If you want to disable notifications after enabling:

1. Go to phone **Settings**
2. **Apps** → **ISP Client Portal**
3. **Notifications** → **Disable**

**Note:** You won't receive important payment reminders if disabled.

   ```
   Settings → Apps → ISP Client Portal → Storage → Clear Data
   ```
   Then login again.

2. **Check Android Version:**
   - Android 13+ requires explicit permission
   - Make sure you're on latest app version

3. **Reinstall App:**
   - Uninstall completely
   - Download latest version
   - Install and allow notifications

4. **Contact Support:**
   - If nothing works, contact your ISP support
   - They can help troubleshoot
