🚀 QUICK START GUIDE:
═══════════════════════════════════════════════════════════════════════

FOR WEB APP (Already Working):
───────────────────────────────────────────────────────────────────────
1. npm run build
2. vercel --prod
3. Share URL with users
   → https://your-portal.vercel.app


FOR ANDROID APK (New):
───────────────────────────────────────────────────────────────────────
1. Install Android Studio
   → https://developer.android.com/studio

2. Install Java JDK 11+
   → https://adoptium.net/

3. Run build script:
   → Double-click: build-apk.bat

4. In Android Studio:
   → Build → Build APK

5. Get APK from:
   → android/app/build/outputs/apk/debug/app-debug.apk

6. Upload to your server

7. Update download page URLs

8. Deploy download page

WEB APP UPDATES:
───────────────────────────────────────────────────────────────────────
1. Make changes to code
2. npm run build
3. vercel --prod
✅ Users get updates automatically (instant)

APK UPDATES:
───────────────────────────────────────────────────────────────────────
1. Make changes to code
2. npm run build
3. npx cap sync
4. Build new APK in Android Studio
5. Upload new APK to server
6. Notify users to download update
⚠️ Users must manually download and install

🐛 COMMON ISSUES:
═══════════════════════════════════════════════════════════════════════

"SDK location not found"
   → Set ANDROID_HOME environment variable

"Gradle sync failed"
   → File → Invalidate Caches → Restart

"Build failed"
   → Check Java version (need 11+)

"White screen on app"
   → Run npm run build before npx cap sync

"Install prompt not showing"
   → Use manual install (Chrome menu → Install app)

More help: See BUILD_APK_GUIDE.md
